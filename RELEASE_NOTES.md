# Release Notes v1.0.0

家計簿の入力がもっと楽に、速くなる PWA 版が完成しました！🎉
スマホからササッと入力して、後でPCにまとめて取り込む「自分専用の入力端末」です。

## ✨ 今回できるようになったこと

- **スマホに最適化**: Pixel 9a 等の画面で、片手でスイスイ入力できるデザイン。
- **手間を減らすアシスト**:
  - 前回のカテゴリや支払い方法を自動で覚えて提案。
  - よく使うカテゴリはワンタップで選べるショートカットを搭載。
  - 連続入力もスムーズ（保存後、金額だけを消してすぐ次へ）。
- **ダークモード対応**: 夜の入力も目に優しい。システム設定連動 ＆ 手動切り替えOK。
- **サクサク操作**: 左右のスワイプでタブを切り替えられます。
- **オフライン/PWA対応**: ホーム画面に追加すれば、電波がなくてもアプリ感覚で起動。
- **PC連携**: 溜まったデータはいつでもCSV出力してPCへ移せます。

## 💡 使い方のヒント

- 外出先でサッと入力 ➡ 週末にCSVを出してPC版システムへ。
- 設定を変えたいときは `src/lib/constants/` 内のファイルをいじるだけでカスタマイズ可能です。
